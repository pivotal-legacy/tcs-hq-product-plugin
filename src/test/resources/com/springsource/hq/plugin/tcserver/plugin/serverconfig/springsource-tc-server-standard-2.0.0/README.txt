SpringSource tc Server
2.0.0.RELEASE
Standard Edition

(SpringSource is a division of VMware(R), Inc.)



You have downloaded the tc Server Standard Edition.   This edition of tc 
Server is made up of the following features:

  * The tc Server application server.
  * Supporting utilities for creating and starting tc Server instances.
  * Templates for creating specific types of tc Server server instances, 
    such as SSL- or cluster-enabled.
